# Keybindings

## Neotree

- `h` - Toggle hidden files
- `shift + h/l` - toggle between tabs
- `a` - Add new file/directory
- `S` - hsplit
- `s` - vsplit


## AstroVim

- ctrl + hjkl - switch splits
- shift + hjkl - switch tabs
- <leader> + p + I - Install LSB via Mason
- <leader> + g + d - Git diff

## Mason

- X - Uninstall package
